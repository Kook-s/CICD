# CI/CD 프로젝트

## 1. 프로젝트 소개
| 2025.08 의성 청년 개발자 프로젝트에서 진행되었던 프로젝트에 적용되었던 CI/CD 부분의 요소만 가져온 프로젝트 입니다.
Handscope 프로젝트는 노인 비율이 높은 의성에서 노인 층의 뇌진환을 사전에 예방하는 프로젝트로 
특정 손 동작을 매핑하여 뇌 질환 위험 성을 측정하여 예방하는 프로젝트 입니다.
